HTML & CSS Quick-Reference Guide

learn about html tags and css properties with examples.

common html tags:

  • <h1> to <h6> — headings of different sizes
  • <p> — defines a paragraph
  • <img> — inserts an image
  • <ul> — unordered list
  • <ol> — ordered list
  • <table> — defines a table
  • <button> — adds a button
  • <div> — used for grouping sections of content
  • <span> — used for inline grouping of elements
  • <code> — displays inline code

example html structure:

        
<h1>this is a heading</h1>
<p>this is a paragraph.</p>
<button>click me</button>

example of a table:

        
<table border="1">
    <tr>
        <th>name</th>
        <th>age</th>
        <th>country</th>
    </tr>
    <tr>
        <td>john</td>
        <td>25</td>
        <td>usa</td>
    </tr>
</table>

common css properties:

  1. background-color — sets the background color
  2. color — sets text color
  3. font-size — controls text size
  4. margin — adds spacing outside an element
  5. padding — adds spacing inside an element
  6. border — defines the border around an element
  7. width — sets element width
  8. height — sets element height

css example:

        
selector {
    color: red; /* sets the text color to red */
    background-color: lightblue; /* sets the background color to light blue */
    font-size: 16px; /* defines the size of the text */
    margin: 10px; /* adds 10px space outside the element */
    padding: 20px; /* adds 20px space inside the element */
    border: 2px solid black; /* creates a solid black border around the element */
    width: 200px; /* sets the width of the element to 200px */
    height: 100px; /* sets the height of the element to 100px */
}
